A multiplayer chase game built with Three.js and PeerJS.

## How to Play
- **Bunny**: Collect 10 carrots before getting caught!
- **Bobcat**: Catch the bunny!
- **Controls**: WASD to move, SHIFT to sprint.

## Deployment (GitHub Pages)
This game is designed to run on **GitHub Pages**.

1.  **Upload** these files to a GitHub repository.
2.  Go to **Settings** > **Pages**.
3.  Select the **main** branch and save.
4.  Wait a moment, and your game will be live at `https://your-username.github.io/repo-name/`.

**Note**: The game uses ES Modules, so it *must* be served via HTTP/HTTPS. It will not work if you just double-click `index.html` on your computer (unless you use a local server). GitHub Pages handles this automatically!
